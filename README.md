# Skill Exchange Project 🚀

Bem-vindo ao **Skill Exchange Project**, uma plataforma projetada para conectar pessoas que desejam trocar habilidades e conhecimentos mutuamente!

## 📌 Visão Geral
Este sistema permite que os usuários se cadastrem informando o que podem ensinar (Habilidade Oferecida) e o que desejam aprender (Habilidade Desejada). O algoritmo então cruza as informações para encontrar os "Matches" perfeitos e os Professores em Potencial.

## 💻 Como Rodar o Projeto Localmente

**1. Ative o ambiente virtual:**
```powershell
.\venv\Scripts\activate
```

**2. Inicie o servidor Django:**
```powershell
python manage.py runserver
```

**3. Acesse no navegador:**
`http://127.0.0.1:8000/`
